# Changelog

## 1.0.0 (2026-03-17)

Initial release.

### Features

- **Device management**: list devices, list AVDs, start emulators
- **Screenshot capture**: take screenshots with intelligent compression, optional save to disk
- **UI tree inspection**: get element hierarchy with bounds, text, resource IDs, and state
- **Touch automation**: tap coordinates, tap elements by selector, swipe, scroll, type text, press keys
- **Element targeting**: find elements by resource-id, text, or content-desc
- **App lifecycle**: install APKs, launch apps, get current activity
- **Logcat integration**: get/clear logs with filtering by package, level, and timestamp
- **Device info**: model, manufacturer, Android version, API level, screen size, DPI
- **File transfer**: pull files from device to local filesystem
- **Shell access**: run arbitrary ADB shell commands
- **Multi-device support**: target specific devices by ID
- **Auto-discovery**: finds Android SDK at default paths or via ANDROID_HOME
